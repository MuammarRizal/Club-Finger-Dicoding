# Latihan Front-end Development

### Dokumentasi

1. Clone repository ini

```bash
    git clone https://github.com/MuammarRizal/Club-Finger-Dicoding.git
```

2. Masuk ke folder Club-Finger

```bash
    cd club-finger
```

3. Install Dependencies

```bash
    $ npm install
```

4. Build App

```bash
    $ npm run build
```

5. Jalankan dalam mode Development

```bash
    $ npm run start-dev
```
